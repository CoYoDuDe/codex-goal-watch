# Security policy

## Scope

This project can send limited keystrokes to a selected GNU screen window. A misclassification is a security issue. Report vulnerabilities privately through GitHub's security advisory flow for this repository; do not include credentials, screen hardcopies, private prompts, server addresses, or live state files.

## Design commitments

- Fail closed for unknown UI, configuration, session, window, or input state.
- Do not publish tokens, keys, authentication files, logs, backups, UUIDs, IP addresses, local paths, personal data, or production terminal captures.
- Treat patches that broaden automatic input acceptance as security-sensitive.

This is an unofficial community project – not affiliated with or endorsed by OpenAI.
